# Portfolio Website

This is a simple portfolio website designed to showcase projects and skills. It includes a clean layout and responsive design to ensure a great user experience across devices.

## Project Structure

The project is organized as follows:

```
portfolio-website
├── src
│   ├── index.html        # Main HTML document
│   ├── styles            # Directory for CSS styles
│   │   └── style.css     # Styles for the portfolio website
│   ├── scripts           # Directory for JavaScript files
│   │   └── app.js        # JavaScript for interactivity
│   └── assets            # Directory for images and fonts
│       ├── images        # Images used in the website
│       └── fonts         # Fonts used in the website
├── README.md             # Project documentation
└── .gitignore            # Files and directories to ignore by Git
```

## Getting Started

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/portfolio-website.git
   ```
2. Navigate to the project directory:
   ```
   cd portfolio-website
   ```
3. Open `src/index.html` in your web browser to view the website.
